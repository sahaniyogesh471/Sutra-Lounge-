# Bot Access Instructions for Sutra Lounge Hetauda

## Allowed Bots

- Googlebot
- Bingbot
- Slurp (Yahoo)
- Baiduspider
- YandexBot

## Rate Limits

Crawl delay: 1-2 seconds between requests

## Disallowed Paths

- /admin - Administrative interface
- /api - API endpoints
- /private - Private resources

## Content Access

Bots may access:
- Public pages (/, /menu, /reservations)
- Static assets (images, CSS, JS)
- Sitemap and robots.txt
